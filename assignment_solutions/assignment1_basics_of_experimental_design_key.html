Solution Key and Grading Rubric

Each question includes a solution key with step-by-step explanations and a grading rubric that outlines how points should be awarded.

Part 1: Key Concepts of Experimental Design

1. Defining Objectives

Solution Key

Experiments are conducted for several primary reasons:

  • To determine principal causes of variation in a response variable.
    • Example: Studying how temperature affects product yield in a chemical reaction.
  • To find conditions for optimal response (max/min).
    • Example: Identifying the ideal fertilizer application rate for maximum crop yield.
  • To compare responses across different conditions.
    • Example: Comparing the effectiveness of two manufacturing processes.
  • To develop a predictive model.
    • Example: Creating a statistical model to forecast machinery failure.

(Source: Dean et al., 2017, Ch. 1)

Grading Rubric (10 points)

  • Clear explanation of all four objectives (6 points)
    • Full points for detailed definitions and correct examples.
    • Deduct 1-2 points for missing or vague descriptions.
  • Relevant examples provided (4 points)
    • Full points if each objective has a valid example.
    • Deduct 1-2 points for incorrect or unclear examples.

2. Experimental vs. Observational Studies

Solution Key

Experimental studies involve controlled manipulation of variables to establish cause-effect relationships. - Example: A factory runs an experiment by adjusting machine settings to measure output quality.

Observational studies involve collecting data without intervention. - Example: Monitoring factory machines to identify trends in product defects without changing settings.

(Source: Dean et al., 2017, Ch. 1)

Grading Rubric (10 points)

  • Definition of experimental vs. observational studies (6 points)
    • 3 points each for clear, correct definitions.
  • Examples related to the factory setting (4 points)
    • Deduct 1-2 points for unclear or missing examples.

3. Basic Techniques: Replication, Blocking, and Randomization

Solution Key

  • Replication: Repeating experimental conditions to increase precision.
    • Example: Testing a drug on multiple patients to estimate treatment effects.
  • Blocking: Grouping experimental units based on similarities to control variability.
    • Example: In an agricultural experiment, blocking plots based on soil type.
  • Randomization: Assigning treatments randomly to reduce bias.
    • Example: Randomly assigning test subjects to treatment and control groups.

(Source: Dean et al., 2017, Ch. 1.1.2–1.1.4)

Grading Rubric (10 points)

  • Clear definitions of replication, blocking, and randomization (6 points)
    • Deduct 1-2 points for missing or unclear definitions.
  • Relevant examples (4 points)
    • Deduct 1-2 points for incorrect or vague examples.

Part 2: Checklist for Planning Experiments

1. Defining Objectives in Experiment Planning

Solution Key

Key steps in experimental planning:

  1. Define clear objectives.
  2. Identify sources of variation.
  3. Choose assignment rules for treatments.
  4. Specify measurements and procedures.
  5. Conduct a pilot experiment.
  6. Define the statistical model.
  7. Plan data analysis.
  8. Determine sample size.
  9. Review and revise the plan.

(Source: Dean et al., 2017, Ch. 2.2)

Grading Rubric (10 points)

  • Accurate list of checklist items (6 points)
    • Full points if all are listed; partial credit for missing or incorrect items.
  • Explanation of defining objectives and identifying variation (4 points)
    • Deduct 1-2 points for incomplete explanations.

2. Purpose of Pilot Experiments

Solution Key

Pilot experiments:

  • Help identify unforeseen challenges.
  • Provide preliminary estimates of variation.
  • Allow refinement of experimental procedures.

(Source: Dean et al., 2017, Ch. 2.2.5)

Grading Rubric (5 points)

  • Accurate explanation of purpose (3 points)
  • Examples or rationale provided (2 points)

3. Hypothetical Scenario: Fertilizer Experiment

Solution Key

  • Define Objectives: Determine which fertilizer maximizes plant growth.
  • Identify Sources of Variation: Soil type, sunlight, watering.
  • Assignment Rule: Randomly assign fertilizers to plots, use blocking if needed.

(Source: Dean et al., 2017, Ch. 2.2.1–2.2.3)

Grading Rubric (10 points)

  • Clear outline of experiment (6 points)
  • Appropriate use of blocking/randomization (4 points)

Part 4: Numerical Application

1. Statistical Model

Solution Key

\[ Y_{ij} = \mu + \tau_i + \beta_j + \epsilon_{ij} \]

where:

  • \(Y_{ij}\) = Response variable
  • \(\mu\) = Grand mean
  • \(\tau_i\) = Treatment effect
  • \(\beta_j\) = Block effect
  • \(\epsilon_{ij}\) = Error term

Grading Rubric (5 points)

  • Correct model equation (3 points)
  • Definitions provided (2 points)

2. ANOVA in R

Solution Key

data <- data.frame(
  Block = factor(rep(1:5, each=3)),
  Treatment = factor(rep(c("A", "B", "C"), times=5)),
  Response = c(10.1, 9.8, 10.2, 10.5, 9.7, 10.3, 10.3, 10.1, 10.4, 10.4, 9.9, 10.0, 10.2, 10.0, 10.1)
)

anova_result <- aov(Response ~ Treatment + Block, data=data)
summary(anova_result)

Grading Rubric (10 points)

  • Correct R code (6 points)
  • Interpretable output (4 points)

3. Interpretation of ANOVA

Solution Key

  • Null Hypothesis (\(H_0\)): No treatment effect.
  • Alternative Hypothesis (\(H_A\)): At least one treatment differs.
  • Decision: Reject \(H_0\) if p-value < 0.05.

Grading Rubric (10 points)

  • Correct hypothesis statements (4 points)
  • Interpretation of p-value (6 points)
