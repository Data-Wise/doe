1.1 Balloon Experiment

(a) Plot inflation time versus balloon color and comment on the results.

R Code:

pacman::p_load(ggplot2)
ggplot2::theme_set(ggplot2::theme_minimal())
# Create data
data <- data.frame(
    TimeOrder = 1:32,
    Color = factor(c(1, 3, 1, 4, 3, 2, 2, 2, 3, 1, 2, 4, 4, 4, 3, 1, 2, 1, 4, 3, 1, 4, 4, 2, 2, 4, 2, 3, 3, 1, 1, 3)),
    InflationTime = c(
        22.0, 24.6, 20.3, 19.8, 24.3, 22.2, 28.5, 25.7, 20.2, 19.6, 28.8, 24.0, 17.1, 19.3, 24.2, 15.8,
        18.3, 17.5, 18.7, 22.9, 16.3, 14.0, 16.6, 18.1, 18.9, 16.0, 20.1, 22.5, 16.0, 19.3, 15.9, 20.3
    )
)

data <- transform(data, Color = factor(Color, levels = c(1, 2, 3, 4), labels = c("Pink", "Yellow", "Orange", "Blue")))
ggplot(data, aes(x = Color, y = InflationTime)) +
    geom_point() +
    geom_boxplot(aes(fill = Color), alpha = 0.5) +
    labs(
        title = "Inflation Time by Balloon Color",
        x = "Balloon Color",
        y = "Inflation Time (seconds)"
    )

Output Plot: The scatter plot shows inflation time by balloon color (pink, yellow, orange, blue). We observe variability among the colors, with blue balloons generally requiring less time and yellow balloons requiring more.

Comment: - Inflation time appears to vary among balloon colors. - Visual evidence suggests color may influence inflation time.

(b) Estimate the mean inflation time for each balloon color and add to the plot.

R Code:

# Calculate means
means <- aggregate(InflationTime ~ Color, data = data, mean)

# Overlay means on plot
ggplot(data, aes(x = Color, y = InflationTime)) +
    geom_point() +
    geom_point(data = means, aes(x = Color, y = InflationTime), color = "red", size = 3) +
    labs(
        title = "Mean Inflation Time by Balloon Color",
        x = "Balloon Color",
        y = "Inflation Time (seconds)"
    ) +
    theme_minimal()

Output Means:

Pink (1) 19.4
Yellow (2) 22.9
Orange (3) 23.1
Blue (4) 18.3

Comment: - The overlay clearly shows differences in mean inflation times. - Blue balloons inflate faster on average, while yellow balloons inflate slower.

(c) Perform an ANOVA test and interpret results.

R Code:

# Perform ANOVA
model <- aov(InflationTime ~ Color, data = data)
summary(model)

R Output:

            Df Sum Sq Mean Sq F value Pr(>F)

Residuals   28  352.4   12.58

Comment: - The p-value is 0.0377, which is less than 0.05. Therefore, we reject \(H_0\) and conclude that balloon color significantly affects inflation time.

(d) Plot data in collection order and discuss assumptions.

R Code:

ggplot(data, aes(x = TimeOrder, y = InflationTime, color = Color)) +
  geom_line() +
  labs(title = "Inflation Time in Collection Order",
       x = "Time Order",
       y = "Inflation Time (seconds)") +
  theme_minimal()

Comment: - The line plot suggests no obvious trends in time order that would indicate violations of independence. - Normality can be verified using residuals from the ANOVA.

(e) Evaluate the analysis and suggest improvements.

Comment: - The analysis is adequate for detecting overall differences among colors. - Improvements: - Check residual plots for homoscedasticity and normality. - Consider pairwise comparisons (e.g., Tukey HSD) for detailed insights.

1.2 Conceptual Question

Answer: - Increasing replicates reduces standard errors of group means, enhancing power. - Power improves the ability to detect smaller differences among group means.

Set 2: Advanced Questions

2.1 Derivation of \(t\)-Statistic for Contrasts in ANOVA

Answer: 1. Variance of \(\sum_{i=1}^a \lambda_i \bar{y}_i = \sigma^2 \sum_{i=1}^a \frac{\lambda_i^2}{N_i}\). 2. Replace \(\sigma^2\) with \(MSE\) as an unbiased estimate. 3. Construct \(t\)-statistic:

\[ t = \frac{\sum_{i=1}^a \lambda_i \bar{y}_i - \sum_{i=1}^a \lambda_i \mu_i}{\sqrt{MSE \sum_{i=1}^a \frac{\lambda_i^2}{N_i}}} \]

2.2 Implications of Assumption Violations

Answer: - Violations of homogeneity affect the validity of \(F\)-tests. - Solutions: 1. Transform data (e.g., logarithmic transformation). 2. Use robust ANOVA methods.
